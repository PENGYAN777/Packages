Extensions of CoolProp library.

There are some .py files:

---------------- plotPython.py: ignore this file

0. installCoolProp.py------------input: none. function: install CoolProp python package. 

1. PT_diagram.py-----------------input: fluid name. function: plot PT diagram for given fluid, including saturation curve.

2. plotContour.py----------------input: fluid name. function: plot contour of Z and Gamma in PT diagram.

3. newIOpairs.py------------------input: fluid name. function: define new input and output pairs between P,T,Z,Gamma

4. readFile.py--------------------input: csv file name. function: read P,T from file and output Z,Gamma in the same file.

5. main.py-----------------------input: none. function: main function.
